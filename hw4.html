Lists Usage
Unordered List An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.

The list items will be marked with bullets (small black circles) by default.
Ordered List An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

The list items will be marked with numbers by default.
Definition List A definition list is used for glossary, vocabulary, questions & answers, etc.This list starts witht the The </dl> and ends with a closing <dl> tag.

Between these two tags, youwill have the pairs of terms and definitions. The terms are presented surrounded by the <dt>and </dt>tags and the definitions by the<dd>and</dd>tags.
